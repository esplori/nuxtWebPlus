### 选择nuxt的原因：

- nuxt开发的项目能被搜索引擎收录，能进行seo。

- 上手简单，语法跟vue基本一致。

- 能正常接入统计跟广告代码。

### 如何启动应用

```bash
# 下载代码
git clone https://gitee.com/wilkwo/nuxt-web-plus.git
# 安装依赖
$ npm install
# 本地启动在 localhost:3000
$ npm run dev
# 部署到服务器，先build，再执行start启动
$ npm run build
# 生成静态项目
$ npm run generate
```
### 注意
- 默认调用官网接口，数据跟官网同步
  
- 如果要使用本地接口，请修改/plugins/baseUrl.js 文件中的baseUrl
